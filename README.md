# Muscafit

Strength training accountability tracker for small groups. Log daily exercises, track activities, leave comments for your training partners, and get AI-powered weekly insights.

## Features

- **Daily exercise logging** with flexible target types (reps, sets, timed, weighted)
- **Activity tracking** for cardio and other sessions
- **Accountability** — see your training partner's progress and leave comments
- **Scheduled email notifications** — evening reminders, morning summaries, weekly reports
- **AI coaching** — weekly insights generated by Claude, analysing your training patterns
- **Goals** — individual and group goals with progress tracking
- **Exercise catalogue** — shared library with custom exercise support

## Tech stack

- Next.js 14 (App Router) + TypeScript
- Tailwind CSS
- better-sqlite3 (file-based, no external DB needed)
- NextAuth.js (credentials provider)
- Resend (transactional email)
- Anthropic Claude API (weekly insights)
- node-cron (scheduled notifications)

## Setup

```bash
# Install dependencies
npm install

# Copy and configure environment variables
cp .env.example .env

# Seed the database with test users and exercises
npm run seed

# Start development server
npm run dev
```

The app runs on **port 3010** by default.
